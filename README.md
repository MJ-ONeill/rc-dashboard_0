A simple orgeniser web app.

Work from home individuals can save links that they regularly use. - instead of a crowded bookmarks tab have only 1 bookmark with everything work related.

Make notes, see schedule and more It saves too your local storage so can open and close without fear of loosing something. - Minimising the number of open apps running freeing up system resources.

Responsive can be used on mobile or PC no need to scroll to the side if want to use on smaller screens.
